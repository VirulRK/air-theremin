# Air-Theremin: Cinematic Harmonics 🎻✨

A contactless, gesture-driven digital audio instrument inspired by the hauntingly beautiful textures of baroque pop and classical symphonies. By bridging MediaPipe's vision models with advanced audio pipelines, this program allows you to sculpt raw acoustic energy out of thin air.

### Key Features:
* **Interactive Laser Strings:** The viewport is divided into a musical matrix. Passing your finger through boundary zones triggers real-time physical bend-and-ripple animations on the screen.
* **Orchestral Wave Synthesis:** Utilizes localized array generation to simulate rich, multi-harmonic acoustic string and piano chords that decay organically.
* **Immersive FX Particles:** Features velocity-sensitive brush tracing and dynamic golden ember particle bursts that react to the speed and intensity of your performance.
